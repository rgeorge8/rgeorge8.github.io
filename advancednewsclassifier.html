Project 3: Advanced News Classifier

Java

Breakdown of the project

In this project, I was tasked with constructing an advanced news classifier utilising GloVe Embedding and Machine Learning.

There were 6 main tasks:

Task 1:

I had to write code that will read data from the files and create the associated Glove objective. Unlike the TF-IDF Embedding in the news classifier, in the advanced news classifier these Glove objectives are word-level embedding (or vectorisation) instead of document-level. So each news article has one single embedding that represents its content.

Task 2:

I had to get the basic information about the news articles (located in the resources\News folder on github). This included coding the relevant get and set methods and writing code to extract the:

-News title
-News content
-News Type (whether it is training or testing data)
-News Label (the label refers to the output or target variable a model tries to predict or classify in machine learning)

Task 3:

Using a BufferedReader to read data from the Glove file line by line and adding different parts of the document to different lists, e.g. its vector representation to a list called listVectors.

Task 4:

Performing text cleaning tasks (text lemmatizatiion and stop word removal) on the data and populating the Glove list. Only non-stop words have a Glove object created.

Task 5:

Creating an array using ND4J (N-Dimensional Arrays for Java). The array is formed by the embeddings of words, which are retrieved from the Glove object list.

Task 6:

The machine learning process consists of an input NDArray, which is simply the document-level embedding, and the output NDArray, which assigns a label to the nth element of the document depending on what group it belongs to. For example, [1,0] if it belongs to the first group or [0,1] if it belongs to the second group. The code is flexible enough to handle more than 2 newsgroups.


We are then left with our expected outcome.
